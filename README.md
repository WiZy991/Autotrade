# AutoTrade - Сайт для импорта автомобилей

Веб-сайт для компании, занимающейся импортом автомобилей из Японии, Кореи и Китая.

## Технологии

- **Backend**: Python 3.x, Flask
- **Frontend**: HTML5, CSS3, JavaScript
- **Стиль**: Минималистичный дизайн

## Установка и запуск

### 1. Установите зависимости

```bash
pip install -r requirements.txt
```

### 2. Запустите приложение

```bash
python app.py
```

### 3. Откройте в браузере

Перейдите по адресу: http://127.0.0.1:5000

## Функциональность

- Главная страница с информацией о компании
- Каталог автомобилей с фильтрацией по странам
- Страницы для каждой страны (Япония, Корея, Китай)
- Детальные страницы автомобилей
- Форма обратной связи
- Страница "О нас"
- Адаптивный дизайн для мобильных устройств
- Современный минималистичный дизайн

## Настройка

Вы можете изменить данные о странах и автомобилях в файле `app.py` в переменных `COUNTRIES` и `CARS`.

## Контакты

Для настройки контактной информации отредактируйте соответствующие разделы в шаблонах.
