# Видео для hero-секции

Поместите сюда ваше видео для фона главной страницы.

## Требования к видео:

1. **Формат:** MP4 или WebM (рекомендуется MP4)
2. **Название файла:** `hero-video.mp4` (или `hero-video.webm`)
3. **Разрешение:** Рекомендуется 1920x1080 (Full HD) или выше
4. **Длительность:** Короткое видео (10-30 секунд), которое будет зациклено
5. **Размер файла:** Старайтесь оптимизировать видео (желательно до 10-20 МБ)

## Как добавить видео:

1. Поместите ваш видео файл в эту папку с именем `hero-video.mp4`
2. Если у вас есть WebM версия, добавьте `hero-video.webm` для лучшей совместимости
3. Видео автоматически появится на фоне главной страницы

## Альтернатива:

Если вы хотите использовать другое имя файла, измените путь в файле `templates/index.html` в секции `<video>`.
