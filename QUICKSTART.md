# Быстрый старт

## Установка и запуск

1. **Установите Python 3.7+** (если еще не установлен)

2. **Установите зависимости:**
   ```bash
   pip install -r requirements.txt
   ```

3. **Запустите приложение:**
   ```bash
   python app.py
   ```

4. **Откройте браузер:**
   Перейдите по адресу: http://127.0.0.1:5000

## Основные страницы

- **Главная** (`/`) - информация о компании и рекомендуемые автомобили
- **Каталог** (`/catalog`) - полный каталог автомобилей с фильтрацией
- **Страны** (`/country/japan`, `/country/korea`, `/country/china`) - автомобили по странам
- **Детали автомобиля** (`/car/<id>`) - подробная информация об автомобиле
- **О нас** (`/about`) - информация о компании
- **Контакты** (`/contact`) - форма обратной связи

## Настройка данных

Все данные о странах и автомобилях находятся в файле `app.py`:
- `COUNTRIES` - информация о странах
- `CARS` - список автомобилей

Вы можете легко изменить эти данные под ваши нужды.

## Изменение контактной информации

Контакты можно изменить в шаблонах:
- `templates/contact.html` - страница контактов
- `templates/base.html` - футер сайта

## Добавление изображений

Для добавления реальных изображений автомобилей:
1. Поместите изображения в папку `static/images/`
2. Обновите пути к изображениям в переменной `CARS` в файле `app.py`

Пример:
```python
'image': url_for('static', filename='images/toyota-camry.jpg')
```
